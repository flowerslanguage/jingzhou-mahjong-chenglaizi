麻将规则：
补充特殊名词：
1、顶果：起完牌后翻一张牌，这张牌就是顶果
2、癞子：顶果+ 1就是癞子（9->1）
3、逞癞子：打掉癞子并从牌头摸一张牌
4、软胡：胡牌时手上有一张癞子（影响结算）
5、硬胡：胡牌时手上没有癞子（影响结算）
6、癞油：逞掉癞子后马上自摸胡牌了（影响结算）
7、黑油：逞掉癞子后马上自摸胡牌了且手上没有癞子（影响结算）
8、软油：逞掉癞子后马上自摸胡牌了且手上有一个癞子（影响结算）
9、暗杠：自摸4张一样的牌（影响结算）
10、直杠：手上3张一样的牌随后别人打给你杠（影响结算）

胡牌规则：
1、只能自摸，不能点炮胡
2、胡牌时最多只能有一个癞子，癞子可充当任意牌
3、胡牌牌型只能是4副顺子或刻子（或杠牌）加一对将，没有7对胡

结算规则：
1、底钱5元，最基础的胡牌叫软胡，得5元，3家各给5元共15元；硬胡（无逞癞子的情况下）得10元，3家各给10元共30元
2、暗杠，摸齐4张相同牌的时候当场给钱，3家各给10元，直杠，打给自己杠的那家给15元，另外2家不给
3、如果手上有一对顶果，别人打了一张顶果，自己碰之后打的人当时给钱15元
4、如果逞一次之后马上摸牌胡牌了（黑油胡牌）翻8倍即40元「10（硬胡）*2（逞的次数*2）*2（黑油）=40」，3家各40共120，如果逞一次之后马上摸牌胡牌了（软油胡牌）翻4倍即20元「5（软胡）＊2（逞的次数＊2）＊2（软油）=20」，3家各20共60
5、如果逞一次之后马上摸牌没有胡牌，但最后又自摸胡牌了，则在软胡或硬胡的基础上*逞的次数，3家都给
6、如果逞了之后到最后都没有胡牌，别人胡牌了，逞了几次就翻几倍赔付给别人多少钱
